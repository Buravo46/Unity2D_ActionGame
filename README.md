Unity2D_ActionGame
==================

###【Unity】Unityの2d機能を使った、キーボードで操作する簡単なアクションゲーム  
- Spriteのコマ送りアニメーション  
- Spriteをアニメーションで上下に動かす  
- スクリプト  
 + Targetのオブジェクトを中心に映して追従するカメラ  
 + Playerをキーボードで上下左右に操作  
 + collision2Dを使った当たり判定  
 + Sceneを替えるためのGUI設置
